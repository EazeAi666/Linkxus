# Linkxus 🔗

**Real-Time Tech, Crypto & AI News Platform**

[![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat&logo=firebase&logoColor=black)](https://firebase.google.com)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Linkxus is a modern, real-time news aggregation platform featuring tech, cryptocurrency, cybersecurity, and university updates. Built with Firebase for authentication, real-time database, and hosting.

## 🚀 Features

- **Real-Time News Feed** - Live updates via Firestore
- **Firebase Authentication** - Google, GitHub, Email/Password, Anonymous
- **SWAL AI Assistant** - Smart Web Analysis Layer for news insights
- **Market Ticker** - Live crypto and stock prices
- **Category Filtering** - Tech, Crypto, Cybersecurity, Universities, Sports
- **Social Features** - Like, comment, share posts
- **Image Uploads** - Firebase Storage integration
- **Responsive Design** - Mobile-first with Tailwind CSS

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| **Frontend** | HTML5, Tailwind CSS, Vanilla JS |
| **Backend** | Firebase (Firestore, Auth, Storage, Hosting) |
| **AI** | Custom SWAL AI integration |
| **Charts** | Chart.js for market data |
| **Icons** | Font Awesome 6 |
| **Fonts** | Inter, JetBrains Mono |

## 📦 Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Firebase CLI](https://firebase.google.com/docs/cli) (`npm install -g firebase-tools`)
- Firebase Account (free tier works)
